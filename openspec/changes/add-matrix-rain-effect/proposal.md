# Change: 添加矩阵字符雨效果

## Why
创建一个经典的"黑客帝国"风格字符雨效果，作为 SymbolMove 项目的第一个核心视觉特效。这将展示项目的核心理念——让字符"动起来"创造视觉世界。

字符雨效果是终端艺术的经典代表，同时也是一个很好的起点，可以作为后续更复杂特效的基础。

## What Changes
- 实现矩阵字符雨效果核心引擎
- 支持可配置的字符集（数字、字母、日文片假名、中文等）
- 支持终端颜色输出（绿色为主，支持渐变效果）
- 支持可调节的雨滴速度和密度
- 提供独立的可执行示例程序

## Impact
- 新增规格: `specs/matrix-rain` - 字符雨效果的完整规格
- 新增代码:
  - `pkg/effects/matrix-rain/` - 核心实现包
  - `cmd/matrix-rain/` - 可执行示例程序
- 依赖: 可能需要引入终端控制库（如 tcell 或 termbox-go）
