# Change: 添加语言切换功能

## Why
当前主界面只使用中文显示,对于国际用户不够友好。添加中英文切换功能可以提升用户体验,让更多用户能够使用符动世界。同时,中英文切换可以让界面更加丰富多样,避免单一语言的单调感。

## What Changes
- 添加语言管理模块,支持中文和英文两种语言
- 添加 `Ctrl+Space` 快捷键在选择器界面切换语言
- 支持翻译所有界面文本:
  - 标题和副标题
  - 操作提示信息
  - 特效名称
  - 特效描述
- 添加用户语言偏好持久化存储(保存到配置文件)
- 在界面上显示当前语言指示器

## Impact
- 影响的功能模块:
  - **NEW**: `pkg/i18n` - 新增语言管理模块
  - **MODIFIED**: `pkg/ui/selector` - 修改选择器支持语言切换
  - **MODIFIED**: `cmd/symbol-move` - 修改主程序初始化语言系统
  - **MODIFIED**: `pkg/effects` - 扩展 Metadata 结构支持多语言
- 影响的文件:
  - 所有特效的 `effect.go` 文件需要添加英文元数据
  - 选择器界面渲染逻辑
  - 主程序启动逻辑
- 用户体验:
  - 用户可以通过 `Ctrl+Space` 随时切换界面语言
  - 语言偏好会被记住,下次启动时自动使用
  - 界面会显示当前语言,方便用户识别
