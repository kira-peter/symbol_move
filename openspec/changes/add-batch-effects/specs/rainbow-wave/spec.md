# 彩虹波浪特效

## 概述

彩虹波浪特效使用彩虹色的水平波浪从左到右滚动，使用 ASCII 字符绘制波浪形状，7种彩虹颜色循环变化。

## 特效 ID

`rainbow-wave`

## 显示名称

彩虹波浪

## 描述

彩虹色的水平波浪从左到右滚动，充满整个屏幕

## 核心特性

### 1. 波浪形状
- 使用正弦函数生成波浪形状
- 多层波浪叠加
- 平滑滚动动画

### 2. 彩虹颜色
- 7种标准彩虹颜色：红、橙、黄、绿、青、蓝、紫
- 颜色渐变过渡
- 循环变化

### 3. ASCII 字符
- 使用波浪字符: ~≈∼≋
- 根据波浪高度选择不同字符
