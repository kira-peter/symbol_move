# 变更提案：批量添加终端特效

## 提案 ID
add-batch-effects

## 概述
为 SymbolMove 项目批量添加 12 个全新的终端视觉特效，丰富特效库并展示插件架构的可扩展性。

## 为什么 (Why)
- **丰富用户体验**：目前仅有矩阵字符雨一个特效，用户选择有限
- **验证架构设计**：通过批量实现验证插件架构的可扩展性和易用性
- **技术探索**：每个特效展示不同的终端渲染技术和算法
- **社区吸引力**：更多酷炫特效可以吸引更多用户和贡献者

## 目标 (What)
实现以下 12 个全新特效：

### 基础特效（简单）
1. **星空闪烁** (starry-sky) - 随机闪烁的星星，模拟夜空
2. **雪花飘落** (snowfall) - 从上到下飘落的雪花
3. **数字瀑布** (digital-waterfall) - 数字从上到下流动

### 动态文字（中等）
4. **波浪文字** (wave-text) - 文字以正弦波形式波动
5. **彩虹波浪** (rainbow-wave) - 彩虹色的波浪动画

### 粒子系统（中等）
6. **粒子爆炸** (particle-burst) - 鼠标点击位置爆炸的粒子效果

### 实用特效（中等）
7. **大字时钟** (big-clock) - 使用大字符显示的时钟

### 高级算法（复杂）
8. **火焰燃烧** (fire-effect) - 模拟火焰燃烧效果
9. **生命游戏** (game-of-life) - Conway's Game of Life 细胞自动机
10. **迷宫生成** (maze-generator) - 实时迷宫生成动画
11. **Plasma 等离子** (plasma) - 彩色等离子效果
12. **音频可视化** (audio-visualizer) - 音频波形/频谱可视化

## 影响 (Impact)

### 新增规格
- 12 个独立的特效规格文档（specs/*/spec.md）
- 每个特效包含需求定义、场景描述、技术约束

### 代码变更
- 新增 12 个特效包：`pkg/effects/*/`
- 每个包包含：核心逻辑、特效接口实现、自动注册
- 可选：独立 CLI 程序 `cmd/*/main.go`

### 架构验证
- 验证特效注册表的可扩展性
- 验证选择器 UI 的动态加载能力
- 验证特效生命周期管理的健壮性

### 文档更新
- README.md 添加所有新特效的描述
- 更新路线图标记已完成项

## 实施策略

### 分批实施
1. **第一批**（简单）：星空闪烁、雪花飘落、数字瀑布
2. **第二批**（中等）：波浪文字、彩虹波浪、粒子爆炸、大字时钟
3. **第三批**（复杂）：火焰燃烧、生命游戏、迷宫生成
4. **第四批**（高级）：Plasma 等离子、音频可视化

### 质量保证
- 每个特效独立测试
- 确保符合 Effect 接口规范
- 性能测试（目标 30+ FPS）
- 终端尺寸适配测试

### 风险控制
- 复杂特效可能需要更多优化时间
- 音频可视化可能需要额外依赖库
- 某些特效在不同终端的兼容性需要测试

## 成功标准
- [ ] 所有 12 个特效成功注册到特效列表
- [ ] 每个特效都能通过选择器 UI 启动
- [ ] 所有特效支持 ESC 键退出
- [ ] 性能达标（CPU 占用 < 30%，FPS ≥ 30）
- [ ] 文档完整更新
- [ ] 通过跨终端测试（Windows Terminal、iTerm2、GNOME Terminal）

## 时间线
- 提案创建：立即
- 规格编写：每个特效 0.5-1 小时
- 实施开发：简单特效 2-3 小时，复杂特效 5-8 小时
- 测试验证：1-2 小时
- 文档更新：1 小时

## 备注
本提案采用批量方式是为了效率，但实际实施仍然遵循一个特效一个规格的原则，确保质量。
