# 配置文件说明

## 配置文件位置

SymbolMove 的用户配置文件保存在:

- **Windows**: `%USERPROFILE%\.symbolmove\config.json`
- **Linux/macOS**: `~/.symbolmove/config.json`

## 配置文件格式

配置文件使用 JSON 格式,示例:

```json
{
  "language": "zh",
  "version": "1.0"
}
```

## 配置项说明

### language (语言设置)

- **类型**: 字符串
- **可选值**:
  - `"zh"` - 中文界面
  - `"en"` - 英文界面
- **默认值**: `"zh"` (中文)
- **说明**: 设置主界面的显示语言,包括标题、提示、特效名称和描述等所有文本

### version (配置版本)

- **类型**: 字符串
- **当前值**: `"1.0"`
- **说明**: 用于配置文件格式的版本控制,便于未来升级时的兼容性处理

## 如何修改配置

### 方法 1: 使用程序内快捷键 (推荐)

1. 运行 `symbol-move.exe`
2. 在主界面按下 `Ctrl+Space` 切换语言
3. 程序会自动保存您的选择到配置文件

### 方法 2: 手动编辑配置文件

1. 打开配置文件 `~/.symbolmove/config.json`
2. 修改 `language` 字段的值
3. 保存文件
4. 重新启动程序即可生效

**注意**: 如果手动编辑导致 JSON 格式错误,程序会忽略配置文件并使用默认值(中文)

## 配置文件的创建

- 首次运行程序时,不需要手动创建配置文件
- 当您第一次使用 `Ctrl+Space` 切换语言时,程序会自动:
  1. 创建 `~/.symbolmove/` 目录(如果不存在)
  2. 创建并保存配置文件

## 重置配置

如需重置所有配置到默认值:

1. 删除配置文件: `~/.symbolmove/config.json`
2. 重新启动程序

程序将使用默认设置(中文界面)
